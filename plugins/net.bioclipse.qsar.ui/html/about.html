About Bioclipse-QSAR

Overview

Bioclipse-QSAR is a feature ( = set of plugins) to work with Quantitative Structure-Activity Relationships (QSAR/QSPR). It contains intuitive graphical editors and wizards that allow for easy setting up a QSAR-ML compliant data set and interact with the Blue Obelisk Descriptor Ontology (BODO).

If you use the Bioclipse-QSAR feature in a scientific project, please cite:

Ola Spjuth, Egon L Willighagen, Rajarshi Guha, Martin Eklund, Jarl ES Wikberg
