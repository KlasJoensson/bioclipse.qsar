QSAR Projects

Bioclipse resources are grouped in Projects in the Bioclipse Navigator; see the page Bioclipse Concepts for a general introduction.

A QSAR Project is required to to work with Bioclipse-QSAR in the GUI. It can be created from the main menu under File > New menu and then selecting QSAR > QSAR Project. When created, a new project appears in the Bioclipse Navigator, and contains the central file (also called Project File), qsar.xml. This is the file that contains all information about the QSAR project, such as the chemical structures, chosen descriptors, responses, metadata etc. and adheres to the QSAR-ML exchange format. Double-clicking the qsar.xml file opens the QSAR Editor.

After a QSAR project is built, it can also contain the files dataset.csv and descriptorValues.xml. These are derived (=generated) files which contains the calculated information from the QSAR project. It is not recommended to edit these files as they are overwritten on every QSAR project build.



The Bioclipse Navigator with a QSAR project open. Note the project file qsar.xml and the derived files dataset.csv and descriptorValues.xml.


The file dataset.csv contains a comma-separated matrix of calculated descriptor values concatenated with the Response variable. Double-click the file to open it in the Matrix Editor.



A small CSV file opened in the Matrix Editor.


The file descriptorValues.xml contains the calculated descriptor values in XML, according to the QSAR-ML format.
