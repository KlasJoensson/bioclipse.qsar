Installing Bioclipse-QSAR

Bioclipse-QSAR is a set of plugins for Bioclipse. There are two ways to obtain a Bioclipse-QSAR package:

Download a pre-built Bioclipse-QSAR package

Please see links available from the Bioclipse-QSAR wiki page.

Download Bioclipse and use the software update option

This guarantees the latest version of Bioclipse and Bioclipse-QSAR.

Bioclipse is available from www.bioclipse.net, see the installation page for more information.

After installing Bioclipse, start it and go to the menu Help > Software Updates. Check the QSAR feature and press finish, and after restart you have Bioclipse-QSAR installed.
